# Replication of *Optimal Transport for Domain Adaptation* (Courty et al., 2015)

This repository presents an independent replication of the experiments from the paper **"Optimal Transport for Domain Adaptation"** by Courty et al. Since the original implementation was not publicly released, all experiments have been reimplemented from scratch using the **Python Optimal Transport (POT)** library together with **Scikit-learn**.

The primary objective of this project is to verify the main conclusion of the paper: **class-regularized optimal transport methods (OT-GL and OT-Laplace) consistently outperform classical subspace-based approaches for unsupervised domain adaptation.**

---

# Experimental Protocol

The implementation follows the evaluation protocol described in the original paper as closely as possible.

- **Classifiers**
  - **Synthetic Two Moons:** SVM with an RBF kernel, whose hyperparameters are selected using 5-fold cross-validation.
  - **Visual adaptation tasks:** 1-Nearest Neighbor (1-NN), as reported in the original work.

- **Sampling Strategy**
  - For the Digits, PIE, and Office-Caltech datasets, the source domain is randomly sub-sampled to **20 samples per class**.
  - The DSLR domain contains fewer images and therefore uses **8 samples per class**.

- **Hyperparameter Selection**
  - Regularization parameters (`reg_e`, `reg_cl`) and the subspace dimension (`k`) are selected using a dedicated validation subset from the target domain, avoiding any contamination of the evaluation set.

- **Repeated Experiments**
  - Every reported result is averaged over **10 independent random trials**.

> **Note**
>
> The original paper does not specify the random seeds or the exact hyperparameter search grids. Consequently, the numerical values obtained in this replication may differ slightly from those reported by the authors. Nevertheless, the relative performance ranking of the methods and the overall conclusions remain consistent with the original publication.

# Conclusions

Despite the absence of the original implementation, fixed random seeds, and complete hyperparameter specifications, this replication successfully reproduces the main findings of the paper.

In particular:

- **OT-GL** and **OT-Laplace** consistently outperform exact OT and entropy-regularized OT.
- Optimal transport effectively reduces domain shift across synthetic, digit, face, and object recognition tasks.
- On high-level DeCAF features, optimal transport acts as a powerful distribution alignment mechanism, correcting residual domain discrepancies that remain after deep feature extraction.

Overall, the experiments confirm the central claim of Courty et al. that incorporating class-aware regularization into optimal transport significantly improves unsupervised domain adaptation performance.
